How it works

01 — Detection
YOLO v4
Each frame is resized to 416x416 and fed into YOLO. The network returns bounding boxes, class IDs, and confidence scores. NMS removes duplicate boxes.
02 — Tracking
Euclidean Distance
Each vehicle is identified by its center point. Between frames, we compute the distance — if it's under 50px, it's the same vehicle and keeps its ID.
03 — Counting
Virtual Line
A vertical line crosses the frame. When a tracked vehicle crosses it, it's added to the counter once — a set of counted IDs prevents duplicates.

from object_detection import ObjectDetection
import cv2, math

od  = ObjectDetection()
cap = cv2.VideoCapture("traffic.mp4")
COUNTING_LINE_Y = 400
tracking_objects, counted_ids = {}, set()
count = 0

while True:
    ret, frame = cap.read()
    if not ret: break
    class_ids, scores, boxes = od.detect(frame)
    for obj_id, (cx, cy) in tracking_objects.items():
        if abs(cy - COUNTING_LINE_Y) < 10 and obj_id not in counted_ids:
            count += 1
            counted_ids.add(obj_id)
